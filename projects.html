Creating my personal portfolio was the final project for the SheCodes Responsive workshop, which I completed in October 2022. I had a lot of fun working on this project; learning to integrate responsive elements to my code for text and images, and learning how to do this on a multiple-page site. This weather app was the final project of the SheCode Plus workshop that I completed in September 2022.

This project used an API to get the current weather forecast in thousands of cities worldwide, and included a second API to get the forecast for the next five days.

Learning about API's was fascinating, and learning how I could include that data into a site I coded was such an incredible feeling! I enjoyed learning how to understand the data returned by the API's, and how to navigate the information to decide what to include in the app, as well as how I wanted it to look.
