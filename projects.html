Recent Projects

Take a look at some of my recent work!

My most recent project is actually this portfolio website!

This was the final project for the SheCodes Responsive workshop, which I completed in October 2022. I had a lot of fun working on this project; learning to integrate responsive elements to my code for text and images, and learning how to do this on a multiple-page site. This project was also about learning how to make websites SEO friendly, which I actually found quite enjoyable to do!

It was a bit challenging at times, to work out a good way to make the site responsive and trying to remember everything I had learnt over the previous few weeks. But I do feel like I learnt a lot and it was a fantastic experience to put everything into practice.

This weather app was the final project of the SheCode Plus workshop that I completed in September 2022.

This project used an API to get the current weather forecast in thousands of cities worldwide, and included a second API to get the forecast for the next five days.

Learning about API's was fascinating, and learning how I could include that data into a site I coded was such an incredible feeling! I enjoyed learning how to understand the data returned by the API's, and how to navigate the information to decide what to include in the app, as well as how I wanted it to look.
